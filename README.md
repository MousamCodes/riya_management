# Riya Management Consultancy Website 🌐

A responsive and modern website built for **Riya Management Consultancy** as part of a freelance project.  
The site highlights the consultancy’s services, expertise, and contact details using a clean and professional design.

---

**🔗 Live Demo:** [View Website](https://riya-management.vercel.app)

## 🚀 Features
- **Responsive Design:** Fully optimized for desktop and mobile screens.
- **Service Showcase:** Dedicated sections to highlight consultancy services.
- **Contact Form:** Simple and functional form for inquiries.
- **Smooth Navigation:** Built with minimalistic UI/UX principles.
- **Custom Styling:** Designed using pure **CSS** for a unique brand identity.

---

## 🛠️ Technologies Used
- **HTML5** – Structure and content.
- **CSS3** – Custom styling and responsiveness.
- **JavaScript (Vanilla JS)** – Basic interactivity and form validation.
